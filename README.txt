SITE LES HÉRITIERS — VERSION SANS DOSSIERS

Cette version est faite spécialement pour GitHub quand tu ne peux pas déposer de dossier.

Pour publier sur GitHub Pages :
1. Dézippe ce fichier.
2. Ouvre le dossier dézippé.
3. Sélectionne TOUS les fichiers à l'intérieur.
4. Envoie-les directement à la racine du dépôt GitHub.
5. Il faut voir index.html, styles.css, script.js et toutes les images au même niveau.

Important :
- Ne mets pas le ZIP directement sur GitHub.
- Ne remets pas les images dans un dossier assets.
- Ne renomme pas les fichiers.
- Les symboles s'appellent : daia-symbol.webp, berk-symbol.webp, etc.
- Les portraits s'appellent : daia-portrait.webp, berk-portrait.webp, etc.
- Les pages d'histoire s'appellent : daia-story-01.webp, taliss-story-08.webp, etc.
